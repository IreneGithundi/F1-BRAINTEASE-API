# User Authentication System

# Phase 1: Authentication

# Overview
The authentication system for the Brain Teaser uses JWS (JSON Web Tokens) with DRF. This provides secure, stateless authentication for API access.

## Key Features
User registration with validation
secure login with JWT tokens
Token refresh mechanism (session last 7 days)
Secure logout with token blacklisting
Custom user model for future F1_specific features
Protection against common security vulnerabilities
